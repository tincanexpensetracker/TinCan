# TINCAN Privacy Policy

A single, self-contained static page (`index.html`) hosted free on GitHub Pages,
used as the privacy-policy URL for the TINCAN app on the Google Play Store.
The logo is embedded inside the HTML, so `index.html` is the **only** file you
need to upload — nothing else to break.

## Publish it (GitHub Pages — free)

1. Create a **new public** repo on GitHub, e.g. `tincan-privacy`.
2. Add the single `index.html` file to it.
3. Repo → **Settings** → **Pages** → Source: **Deploy from a branch** →
   Branch: `main`, folder: `/ (root)` → **Save**.
4. Wait ~1 minute. Your live URL will be:
   `https://<your-github-username>.github.io/tincan-privacy/`
5. Paste that URL into **Play Console → App content → Privacy policy**.

## Updating

Edit `index.html`, change the "Last updated" date, commit & push. Pages
redeploys automatically.
